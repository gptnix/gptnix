## 🚀 **Deploy u 5 Minuta**

### **1. Raspakuj ZIP**
```bash
unzip gptnix-backend-v4.1-PRODUCTION.zip
cd gptnix-v4.1
```

### **2. Deploy na Cloud Run**
```bash
# Build
gcloud builds submit --tag gcr.io/YOUR_PROJECT_ID/gptnix-backend:v4.1

# Deploy
gcloud run deploy gptnix-backend \
  --image gcr.io/YOUR_PROJECT_ID/gptnix-backend:v4.1 \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated \
  --memory 1Gi \
  --cpu 1 \
  --timeout 300s
```

### **3. Set Environment Variables**
```bash
# Kopiraj sve env vars iz V4 deployments
gcloud run services update gptnix-backend \
  --region us-central1 \
  --update-env-vars DEEPSEEK_API_KEY=xxx,OPENAI_API_KEY=xxx,...
```

### **4. Quick Test**
```bash
# Health check
curl https://YOUR_SERVICE_URL/health

# Smoke test (optional)
node smoke-test.js https://YOUR_SERVICE_URL YOUR_TOKEN
```
