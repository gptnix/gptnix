**2. Chunk Garbage Filter** (rag.js)
```javascript
// V4.3: Filter binary garbage chunks
const chunks = rawChunks.filter(chunk => {
  const printable = (chunk.match(/[\x20-\x7E\n\r\t]/g) || []).length;
  const ratio = printable / chunk.length;
  
  if (ratio < 0.7) {
    console.log('⚠️ Skipping chunk: only ${ratio}% printable');
    return false;  // SKIP garbage!
  }
  return true;
});
```

**Što radi:**
- ✅ Count printable characters
- ✅ Skip if <70% printable
- ✅ Log what's skipped
- ✅ Error if NO clean chunks
