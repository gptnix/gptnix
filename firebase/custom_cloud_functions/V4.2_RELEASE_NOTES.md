## 🚀 **Deployment Instructions**

### **1. Raspakuj ZIP**
```bash
unzip gptnix-backend-v4.2-PRODUCTION.zip
cd gptnix-v4.2
```

### **2. Deploy to Cloud Run**
```bash
# Build
gcloud builds submit --tag gcr.io/YOUR_PROJECT_ID/gptnix-v4.2

# Deploy
gcloud run deploy gptnix-backend \
  --image gcr.io/YOUR_PROJECT_ID/gptnix-v4.2 \
  --platform managed \
  --region us-central1 \
  --memory 1Gi \
  --timeout 300s
```

### **3. Set Environment Variables**
```bash
gcloud run services update gptnix-backend \
  --region us-central1 \
  --update-env-vars DEEPSEEK_API_KEY=xxx,OPENAI_API_KEY=xxx,...
```

### **4. Verify Deployment**
```bash
# Check logs for V4.2 markers
gcloud run logs read gptnix-backend --limit 100 | grep "V4.2"

# Expected output:
# 📄 [V4.2-DOC-DETECT] File extension found in message
# 📄 [V4.2-ROUTER] File extension detected, forcing RAG plan
```
